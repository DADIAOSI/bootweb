this is Ajax !

  • 数据显示Tab是通过一些常用的label来显示企业的详细信息
  • 表格统计Tab是通过一些经过服务器处理的统计数据形成的表格数据，包括数据库中的源数据表，也可以利用chart图表进行绘制和显示折线图、饼状图等等
  • 查看地图Tab是用来进行企业地点、物资、危险源、事故模拟、救援路线规划等操作的视图，利用Arcgis API for javascript来进行地图上的操作
  • 帮助说明Tab是用来演示每一个子系统的操作过程，以及输入参数的说明
  • 左侧主要工具栏中需要进行参数设置的时候可以用两种方式进行输入
    • 一种是弹出一个对话框进行信息输入
    • 在左侧边框的右边弹出一个DIV，在这其中进行参数输入
  • 用到的JavaScript工具
    • easyUI框架，搭建网站页面布局和相关表格控件
    • 百度eCharts JavaScript，用来进行相关统计图表的显示，例如折线图、饼图、柱状图等等
    • Arcgis API for JavaScript用来进行地图的显示和相关操作
    • jQuery用来选择和操控网页DOM对象，表单数据的提交，以及动画效果的实现
    • 其他待定
  • 图标引用问题：在data-options="iconCls:'icon-application-cascade'"中的icon-application-cascade对应的是icon.css文件中的样式，创建样式并引入图片，在data-options中就可以使用该样式名称了
